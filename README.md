# wx-shop-demo

## 目录结构

|   目录名   |       作用       |
| :--------: | :--------------: |
| components |     存放组件     |
|   icons    |    tab栏图标     |
|   pages    |      页面       |
|   request  | 自己的接口帮助库 |
|   styles   |   存放公共样式   |
|   utils    |   工具库   |
