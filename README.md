# 仿微信商城

## 1.目录结构

|   目录名   |       作用       |
| :--------: | :--------------: |
| components |     存放组件     |
|   icons    |    tab栏图标     |
|   pages    |      页面       |
|   request  | 自己的接口帮助库 |
|   styles   |   存放公共样式   |
|   utils    |   工具库   |



## 2.页面

|   页面名称   |     名称     |
| :----------: | :----------: |
|     首页     |    index     |
|   分类页面   |   category   |
| 商品列表页面 |  goods_list  |
| 商品详情页面 | goods_detail |
|  购物车页面  |     cart     |
|   收藏页面   |   collect    |
|   订单页面   |    order     |
|   搜索页面   |    search    |
| 个人中心页面 |     user     |
| 意见反馈页面 |   feedback   |
|   登录页面   |    login     |
|   授权页面   |     auth     |
|   结算页面   |     pay      |
